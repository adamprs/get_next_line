# get_next_line

## Description

get_next_line consists of implementing a function that reads and returns a line from a file descriptor.

The goal of this project is to understand file I/O in C and the use of static variables. The function is designed to be called repeatedly, returning one line at a time from a file or standard input.

Each call reads just enough data to return the next line, including the newline character when present, until the end of the file or an error occurs.
